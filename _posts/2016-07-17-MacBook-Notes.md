#### Keyboard Windows to Mac Shortcuts ####

Control - Ctrl  
Option - Alt  
Command (Four Leaf Clover) - Windows  
Delete - Backspace  
Return - Enter  

Cmd + W             : Close  
Cmd + Z             : Undo  
Cmd + Q             : Quit App  
Cmd + F             : Search 
Cmd + Opt + D       : Hide / Show Dock  
Cmd + Opt + 3       : Take Screenshot
Cmd + Opt + Esc     : Force Quit  
Cmd + Comma         : Open Preferences  
Cmd + Period        : Cancel an operation  
Cmd + Shift + Q     : Log Out
